# Работа с агентом требует доказательств

Пользователь уже увидел на реальном кейсе, что зелёные unit-тесты не доказывают готовность внешнего сценария. Следующие уроки можно строить от этой базы: агент должен не только написать код, но и принести проверяемое доказательство через GitHub issue, push и E2E.
