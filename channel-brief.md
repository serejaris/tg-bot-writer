# Профиль канала

> Блог «Рецепт дня» — семейное меню с учётом аргентинских реалий и сезонности.

## Канал

- **Название:** Рецепт дня / Menú del día
- **@username:** `[TODO: указать]`
- **Ссылка:** `[TODO: указать]`

## Аудитория

- Русскоязычные семьи в Аргентине (Buenos Aires и крупные города).
- 2 взрослых + 2 детей — формат рассчитан на 4 порции.
- Нужны простые, доступные рецепты из продуктов Coto / Carrefour / Disco и feria.

## Темы

- Ежедневные рецепты: **завтрак + обед** (ужин не публикуем).
- Сезонные продукты Аргентины (зима: июнь–август).
- Списки покупок раз в неделю.
- Лайфхаки: feria vs super, batch-prep в воскресенье, viandas для детей.

## Тон и стиль

- Обращение: **ты**
- Длина: средний пост (~800–1200 зн.) — два блока (завтрак + обед) + короткий совет
- Эмодзи: умеренно (🍳 🥣 🛒 🌧️)
- Юмор: лёгкий, бытовой («пока дети одеваются — я уже реву яйца»)

### Формат ежедневного поста

```
🍳 **Завтрак: [RU] / [ES]** — день [N], неделя [W]

[1 строка-хук: быстро / из запасов / дети едят]

⏱ **Готовка:** X мин | **Порций:** 4

**Ингредиенты:**
`producto — cantidad`
`…`

**Шаги:** (макс. 5–6)
1. …

🥣 **Обед: [RU] / [ES]**
(тот же формат)

💡 ||Совет: …|| ← spoiler в Telegram

🔗 Завтра: [блюдо] | Список недели: [ссылка на пост]
```

- Дублируй продукты: **русское + испанское** (`творог / ricota`).
- Ингредиенты — в `моноширинном` для выравнивания.
- Длинный рецепт (>1500 зн.) — карточка в канале + полная версия в Telegraph.

### Формат еженедельного поста (воскресенье)

```

📋 Меню недели: [ссылка]

**Verdulería** ☐ …

**Carnicería** ☐ …
**Almacén** ☐ …

**Уже дома — не покупать:** …

⏰ **Prep domingo (30–45 мин):** …

Понедельник: **[завтрак]**
```

## Ограничения

- Не публикуем ужин.
- Не выдумываем цены — только «примерно» или `[TODO: актуальная цена]`.
- Аллергии детей: `[TODO: указать, если есть]`
- Без экзотических ингредиентов, которых нет в обычном супере.

## Расписание

- **Ежедневно:** пост с завтраком и обедом на сегодня (утро, ~8:00 ART)
- **Воскресенье:** список покупок на следующую неделю (~10:00 ART)
- Дни без публикации: нет (ежедневный формат)
